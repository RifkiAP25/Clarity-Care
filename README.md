# Clarity-Care
Clarity Care is a web based system prototype designed to manage a pharmacy. This application provides feature depend on the role, admin, aphotecary and customer. The web is indonesian languange based.
This project is made by team of 5 people.
pick the role first, and then login or made an account, and you can try all the feature.
